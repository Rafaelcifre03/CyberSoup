An amazing project by Borja, Rafael, George and Alejandro.

## Run Locally

```bash
  git clone https://github.com/giftimie/cybersoup/
```

Go to the project directory

```bash
  cd cybersoup
```

Install dependencies

```bash
  composer install
```

Start the server

```bash
  XAMPP or any other web server
```
